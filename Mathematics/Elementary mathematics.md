# 函数

## 函数的定义

函数（function）是数学中的一个基本概念，描述了两个集合之间的一种对应关系。

### 定义

设A、B是非空的数集，如果按照某种确定的对应关系f，使对于集合A中的任意一个数x，在集合B中都有唯一确定的数f(x)和它对应，那么就称f：A→B为从集合A到集合B的一个函数（function），记作：
- y = f(x)，x ∈ A

其中：
- x叫做**自变量**，x的取值范围A叫做函数的**定义域**（domain）
- 与x的值相对应的y值叫做**函数值**，函数值的集合{f(x)|x ∈ A}叫做函数的**值域**（range）
- f叫做**对应法则**或**映射法则**

### 函数的三要素

1. **定义域**：自变量x的取值范围
2. **对应法则**：f，表示x与y之间的对应关系
3. **值域**：函数值的集合

### 函数的表示方法

1. **解析法**：用数学表达式表示函数关系，如 y = 2x + 1
2. **列表法**：用表格列出函数值
3. **图像法**：用函数图像表示函数关系

## 五种基本初等函数

基本初等函数是数学中最基础的函数类型，其他函数都可以由这些基本初等函数经过有限次的四则运算和复合运算得到。

### 1. 幂函数

**定义**：形如 y = x^a（a为常数）的函数叫做幂函数。

**一般形式**：y = x^a，其中a ∈ R

| 幂函数类型      | 示例  | 定义域 | 值域   | 奇偶性   | 单调性 (在第一象限) |
| --------------- | ----- | ------ | ------ | -------- | ------------------- |
| α 为正奇数      | y=x3  | R      | R      | 奇函数   | 增                  |
| α 为正偶数      | y=x2  | R      | [0,+∞) | 偶函数   | 增                  |
| α 为负奇数      | y=x−1 | x=0   | y=0   | 奇函数   | 减                  |
| α 为负偶数      | y=x−2 | x=0   | (0,+∞) | 偶函数   | 减                  |
| α 为分数 (根号) | y=x   | [0,+∞) | [0,+∞) | 非奇非偶 | 增                  |

- 周期性 (Periodicity)

  - **结论：** 所有的幂函数（在 $\alpha \neq 0$ 时）都**不是周期函数**。
    - 其图像不会重复出现，这与三角函数有本质区别。

  ------

-  有界性 (Boundedness)

  - **无界性：** 大多数幂函数在全定义域上是无界的。例如 $y=x^2$ 当 $x \to \infty$ 时，$y \to \infty$。
  - **局部有界：** * 当 $\alpha < 0$ 时，在 $(0, +\infty)$ 内，$y > 0$，有下界但无上界。
    - 当 $\alpha > 0$ 时，在特定区间（如 $[a, b]$）内是有界的。

**常见类型**：

- 当 a = 1 时，y = x（正比例函数）
- 当 a = 2 时，y = x²（二次函数）
- 当 a = -1 时，y = 1/x（反比例函数）
- 当 a = 1/2 时，y = √x（平方根函数）

**性质**：
- 定义域：根据a的值而定，通常为(0, +∞)或R
- 图像都经过点(1, 1)
- 当a > 0时，函数在(0, +∞)上单调递增
- 当a < 0时，函数在(0, +∞)上单调递减

### 2. 指数函数

**定义**：形如 y = a^x（a > 0 且 a ≠ 1）的函数叫做指数函数。

**一般形式**：y = a^x，其中 a > 0，a ≠ 1

![[image-20260119153735318.png]]

**性质**：
- 定义域：R（全体实数）
- 值域：(0, +∞)
- 图像都经过点(0, 1)
- 当 a > 1 时，函数单调递增
- 当 0 < a < 1 时，函数单调递减
- 以x轴为水平渐近线

**特殊值**：
- 当 a = e（自然常数，约2.718）时，y = e^x 称为自然指数函数
- 当 a = 10 时，y = 10^x 称为常用指数函数

### 3. 对数函数

**定义**：形如 y = log_a(x)（a > 0 且 a ≠ 1）的函数叫做对数函数。

**一般形式**：y = log_a(x)，其中 a > 0，a ≠ 1

**与指数函数的关系**：对数函数是指数函数的反函数
- 如果 y = a^x，则 x = log_a(y)

![[image-20260119154113102.png]]

**性质**：
- 定义域：(0, +∞)
- 值域：R（全体实数）
- 图像都经过点(1, 0)
- 当 a > 1 时，函数单调递增
- 当 0 < a < 1 时，函数单调递减
- 以y轴为垂直渐近线

**特殊类型**：
- **自然对数**：y = ln(x) = log_e(x)，以e为底
- **常用对数**：y = lg(x) = log₁₀(x)，以10为底

**对数运算法则**：
- log_a(MN) = log_a(M) + log_a(N)
- log_a(M/N) = log_a(M) - log_a(N)
- log_a(M^n) = n·log_a(M)
- log_a(b) = 1/log_b(a)（换底公式）

### 4. 三角函数

**定义**：三角函数是描述角度与直角三角形边长之间关系的函数。

**基本三角函数**：

1. **正弦函数**：y = sin(x)
   - 定义域：R
   - 值域：[-1, 1]
   - 周期：2π
   - 奇函数：sin(-x) = -sin(x)

2. **余弦函数**：y = cos(x)
   - 定义域：R
   - 值域：[-1, 1]
   - 周期：2π
   - 偶函数：cos(-x) = cos(x)

3. **正切函数**：y = tan(x) = sin(x)/cos(x)
   - 定义域：{x | x ≠ π/2 + kπ, k ∈ Z}
   - 值域：R
   - 周期：π
   - 奇函数：tan(-x) = -tan(x)

4. **余切函数**：y = cot(x) = cos(x)/sin(x)
   - 定义域：{x | x ≠ kπ, k ∈ Z}
   - 值域：R
   - 周期：π

5. **正割函数**：y = sec(x) = 1/cos(x)
6. **余割函数**：y = csc(x) = 1/sin(x)

**基本关系式**：
- sin²(x) + cos²(x) = 1（平方和公式）
- tan(x) = sin(x)/cos(x)
- 1 + tan²(x) = sec²(x)
- 1 + cot²(x) = csc²(x)

### 5. 反三角函数

**定义**：反三角函数是三角函数的反函数，用于求已知三角函数值对应的角度。

**基本反三角函数**：

1. **反正弦函数**：y = arcsin(x) 或 y = sin⁻¹(x)
   - 定义域：[-1, 1]
   - 值域：[-π/2, π/2]
   - 单调递增

2. **反余弦函数**：y = arccos(x) 或 y = cos⁻¹(x)
   - 定义域：[-1, 1]
   - 值域：[0, π]
   - 单调递减

3. **反正切函数**：y = arctan(x) 或 y = tan⁻¹(x)
   - 定义域：R
   - 值域：(-π/2, π/2)
   - 单调递增
   - 以 y = π/2 和 y = -π/2 为水平渐近线

4. **反余切函数**：y = arccot(x) 或 y = cot⁻¹(x)
   - 定义域：R
   - 值域：(0, π)
   - 单调递减

**基本关系**：
- arcsin(x) + arccos(x) = π/2（当 x ∈ [-1, 1]）
- arctan(x) + arccot(x) = π/2（当 x ∈ R）

## 初等函数

由基本初等函数经过有限次的四则运算（加、减、乘、除）和有限次的函数复合步骤所构成，并可用一个解析式表示的函数，称为**初等函数**。

例如：
- y = x² + sin(x)（幂函数与三角函数的和）
- y = e^x · ln(x)（指数函数与对数函数的乘积）
- y = sin(e^x)（指数函数与三角函数的复合）
